# AI Team Theme — Cool Current (Creative)

## Overview
This is the official brand identity for the **AI Team**. It must be followed strictly for all creative work, Confluence pages, dashboards, presentations, and UI.

## Logo
- Stylized **"A"** with cyan-to-purple gradient
- Always use the gradient version for headers and key elements

## Color Palette

### Primary Logo Colors
| Name          | Hex       | Usage                          |
|---------------|-----------|--------------------------------|
| Deep Navy     | #1A2080   | Primary dark, headers          |
| Bright Blue   | #6288FC   | Accents                        |
| Cyan          | #078CEA   | Highlights                     |
| Light Blue    | #5C96FC   | Secondary                      |
| Purple        | #3D2CFE   | Creative elements              |
| Teal          | #05D7CE   | Success / positive             |
| Lavender      | #8B76FC   | Soft accents                   |
| Light Purple  | #A98EFC   | Backgrounds                    |

### Cool Current Gradient
- From: `#271F7F`
- To: `#00FFD4`
- Use for: Logo, headings, CTAs, hero sections

## Confluence Design System Guidelines
- Primary Button: `color.button.primary`
- Page Header: `color.surface.nav`
- Sidebar Active: `color.sidebar.selected`
- Info Panel: `color.feedback.info`
- Data Table Header: `color.table.header`

**Note:** Any change to the palette requires approval from the Design Lead.

## Usage Rules
- Always use the colors above — do not invent new ones.
- Prefer the Cool Current gradient for important visual elements.
- Maintain high contrast.
- When creating Confluence pages, dashboards, or HTML, follow the design token names where possible.
